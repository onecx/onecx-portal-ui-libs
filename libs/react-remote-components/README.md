# @onecx/react-remote-components

`@onecx/react-remote-components` manages remote React components, enabling dynamic component loading and integration within OneCX applications.
More information about OneCX libraries can be found in the [OneCX documentation](https://onecx.github.io/docs/documentation/current/onecx-portal-ui-libs/index.html).

## Installation

```bash
npm install @onecx/react-remote-components
```

## Additional Commands
- `npx nx run react-remote-components:build` - Builds the library and outputs the result to the `dist` folder.
- `npx nx run react-remote-components:test` - Runs the unit tests for the library.
- `npx nx run react-remote-components:lint` - Lints the library's codebase.
- `npx nx run react-remote-components:release` - Releases a new version of the library to npm, following semantic versioning guidelines.
